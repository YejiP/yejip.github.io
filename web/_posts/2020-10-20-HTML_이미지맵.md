# HTML IMAGEMAP 

- Imagemap을 이용하면 하나의 이미지에 여러 개의 링크를 걸 수 있다. 다음과 같이 사용한다.  

  ```html
  <img src = "pin.jpg" usemap = ""#water">
  <map name = "water">
      <area shape="rect" coords="220,10,380,200" href ="link" target="_self">
  </map>
  ```
